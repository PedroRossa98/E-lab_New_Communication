Estimating pi (π) using Monte Carlo Simulation

With this HTML, you can perform an experiment remotely, that is, without the need to be present physically in the Laboratory. You will use the Monte Carlo Method to estimate the value of Pi. In order to do that, just fill the input with values that you want.
