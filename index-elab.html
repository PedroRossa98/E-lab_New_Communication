Monte Carlo

Remote Experiment


Estimating pi (π) using Monte Carlo Simulation

With this HTML, you can perform an experiment remotely, i.e, without the need to be present physically into the Laboratory. You will use the Monte Carlo Method to estimate the value of Pi. In order to do that, just fill the input to values that you want and make sure to run the flask server.
The main idea behind this method can be found here. Do not forget to have fun!!!
